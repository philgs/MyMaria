# MyMaria PowerShell Module

## Description

A simple, cross-platform PowerShell module for working with MySQL and MariaDB. This module requires
PowerShell 7.

## Basic Usage

### Importing the Module

To import the module, ensure that the path where the files reside is included in the
`$Env:PSModulePath` variable. After validating that the module is resolvable, issue the following
command...

```powershell
Import-Module MyMaria
```
